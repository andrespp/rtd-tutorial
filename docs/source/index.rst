Documentação do DW-BR
=====================

**DW-BR** é uma armazém de dados públicos, que utiliza modelagem dimensional e
ferramentas de *big data*  com objetivo de oferecer acesso integrado, simples e
intuitivo a conjuntos de dados públicos brasileiros.

Veja a Seção :doc:`usage` para mais informações sobre a instalação e utilização
da ferramenta.

Detalhes sobre a modelagem, origem dos dados, metadados e outras informações
técnicas estão disponíveis na Seção :doc:`model`.

.. note::

   Este projeto está em fase inicial de desenvolvimento.
